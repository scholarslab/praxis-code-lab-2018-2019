Our Epic Las Casas Database Plan
by Catherine, Mathilda, & Chris

For our homework this week, Catherine decided to commandeer the opportunity to force her fellow fellows to start working on her digital edition of Bartolomé de las Casas' 1552 treatises.

The goal was to get Brevísima relación de la destrucción de las Indias, or rather the public-domain 1689 English translation of it (about which Catherine has a soapbox but we can talk about that on our own time), from Project Gutenberg in order to extract all the proper nouns from it, i.e. toponyms and names. English-language texts from this time period happen to italicize proper nouns so we thought we'd pull the italicized snippets and de-dupe the list. That list of toponyms and names could then serve as a basis for a set of encyclopedia entries from which the digital edition could draw (i.e. click a person's name in the text, go to their page to learn more).

Our (theoretical) workflow looked like this:

  1. Scrape Project Gutenberg for the HTML version of the text, 'cause we need those italic tags.
  2. Use Beautiful Soup to get all the italic tags.
  3. De-dupe the list, using maybe the dedupe library.
  4. Would be cool if we made lil dictionaries of toponyms vs. names but like who has time.
  5. Save results to CSV.
  6. Write up in HTML. Push to Github. Party.
